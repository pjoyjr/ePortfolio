Burmese American Community Institute Project

Mentor/Mentee Pairing Website

Site was ran using XAMPP to host over the localhost. The database was created and managed using phpMyAdmin. The site was formatted using Bootstrap and custom CSS. php, HTML5, CSS, JAvascript and AJAX were all used to create the website. I was team manager for the project and had a team of 3 others (Zaid Saeed, Mik Kreilis, and Joshua Smith).
link to all code: https://github.com/pjoyjr/BACI

User Access

Homepage and Signing Up

How to sign up a user, fill out the register form and activate an account from an email.

User Dashboard

Editing Profile, Uploading Files, User Pairings, and View Mentors if you are a Mentee or View Mentees if you are a mentor.

Coordinator Access

Coordinator's also have access to all basic User Features.

Viewing/Editing Users

Searching activated user list based on certain criteria and editing user that user's information if needed.

Checking User Pairings

Adding a new Mentor/Mentee pairing. Seeing all current and ended/denied pairings and any pairings pending approval.

Configure Countries, States and Degree List

How to add, edit, or delete select registration information from the database.

Administator Access

Administator's also have access to all User and Coordinator Features.

Create User/Coordinator

Administator has the ability to create an active account with user or coordinator privileges.

Delete User/Coordinator

Administator also has the ability to delete any account. Account is still saved in database just marked hidden to all other users.

Download Tables from Database

Saving the data from several tables in the database and saved as a .csv
